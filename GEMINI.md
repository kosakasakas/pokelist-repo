## プロジェクト概要

このプロジェクト「Pokelist Tools」は、ポケモンプレイヤー向けに設計されたウェブアプリケーションです。ゲームプレイを支援するためのツール群を提供します。主な機能は以下の通りです。

*   ポケモン、わざ、とくせい、アイテムを閲覧できる総合的な図鑑
*   ダメージ計算ツール
*   すばやさ比較ツール

このアプリケーションは、フロントエンドにHTML、Bootstrap、そして素のJavaScriptを使用しています。また、豊富なポケモンデータと対戦シミュレーションロジックのために、gitサブモジュールとして `pokemon-showdown` プロジェクトを活用しています。

## ビルドと実行

プロジェクトのセットアップと実行には、以下のコマンドが不可欠です。

*   **`npm install`**: プロジェクトの依存関係をインストールします。このコマンドは、`pokemon-showdown` サブモジュールの依存関係も自動的にインストールします。
*   **`npm run setup`**: `pokemon-showdown` のgitサブモジュールを初期化・更新します。リポジトリを初めてクローンした後に実行する必要があります。
*   **`npm run build:showdown`**: `pokemon-showdown` サブモジュールをビルドします。これは、プロジェクト独自のデータをビルドする前に必要なステップです。
*   **`npm run build:data`**: `pokemon-showdown` サブモジュールからアプリケーションに必要なJSONデータを生成します。出力は `public/db/` に配置されます。
*   **`npm run server`**: `public` ディレクトリを配信するローカル開発サーバーを起動し、ブラウザでツールにアクセスできるようにします。

一般的な作業フローは以下の通りです。
1. `npm run setup`
2. `npm install`
3. `npm run build:showdown`
4. `npm run build:data`
5. `npm run server`

## 開発規約

*   **コアロジックのためのサブモジュール**: このプロジェクトは、すべてのコアとなるポケモンデータとゲームの仕組みを `pokemon-showdown` サブモジュールに依存しています。コアデータへの変更や更新は、サブモジュール内で行う必要があります。
*   **データ処理**: `scripts/export-champions-calc-data.js` スクリプトは、`pokemon-showdown` サブモジュールからデータを抽出し、フロントエンドが利用できる形式に変換するために使用されます。このスクリプトは、サブモジュール内の生データとアプリケーションのデータニーズとの間の橋渡しをします。
*   **静的なフロントエンド**: フロントエンドは `public` ディレクトリにある静的なHTML、CSS、JavaScriptファイルで構成されています。複雑なフロントエンドフレームワークは使用されていません。
*   **多言語化（日本語対応）**: このプロジェクトには日本語翻訳の仕組みが含まれており、`public/db/` にある上書きファイルや翻訳漏れファイルでローカライズされたテキストを管理しています。
*   **翻訳のデータ主導**: ドメイン翻訳（タイプ名・技分類・対象・フラグ・タグ・名称など）はUIへハードコードせず、`scripts/export-champions-calc-data.js` で生成される `public/db/champions-ja-translations.json` を参照します。
